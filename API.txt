### Calc
**HTTP-метод:** POST   
**URL:** /calc/
**Описание:** Выполняет расчет стоимости посылки.  
**Параметры запроса:** 
```
{
    "lang": str,
    "name": str,
    "height": float,
    "width": float,
    "depth": float,
    "weight": int,
    "is_safe": bool
}
```

**Описание параметров запроса:**  
- name – (строка) имя доставщика.
- height – (вещественное число) содержит вещественное число с высотой посылки.
- width – (вещественное число) содержит вещественное число с шириной посылки.
- depth – (вещественное число) содержит вещественное число с глубиной посылки.
- weight – (вещественное число) содержит вещественное число с весом посылки.
- is_safe – (логическое значение) содержит логическое значение с информацией о застрахованности посылки.

**HTTP-статус ответа:** 200  
**Состояние:** Запрос завершен успешно.  
**Параметры ответа:** 
```
{
    "price": float,
    "is_safe": str
}
```

**Описание параметров ответа:**  
- price (строка) – цена доставки посылки.
- is_safe (строка) – возвращает строку с информацией о страховке.



### PostCode
**HTTP-метод:** POST   
**URL:** /postcode/
**Описание:** Выполняет поиск адреса.  
**Параметры запроса:** 
```
{
    "address": str
}
```

**Описание параметров запроса:**  
- address - (строка) содержит строку с указанием адреса доставки.


**HTTP-статус ответа:** 200  
**Состояние:** Запрос завершен успешно.  
**Параметры ответа:** 
```
{
	"full_address": str,
	"postcode": str,
	"timezone": str
}
```

**Описание параметров ответа:**  
- full_address – (строка) содержит строку с полным адресом доставки.
- postcode – (строка) содержит строку с почтовым индексом.
- timezone – (строка) содержит строку с часовым поясом.
